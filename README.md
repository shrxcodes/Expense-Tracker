# Expense Tracker 
A simple and beginner-friendly expense tracker built using **HTML, CSS, and JavaScript**.  
This application helps users track their income, expenses, and remaining balance in real time.

# Features
- Set total income
- Add daily expenses
- Automatically calculates total expenses
- Displays remaining balance (savings)
- Clean and user-friendly interface

# Tech Stack
- HTML
- CSS
- JavaScript

# How It Works
1. Enter your total income and set it.
2. Add expenses with name and amount.
3. The app updates:
   - Total expenses
   - Remaining balance automatically
